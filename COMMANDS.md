# Development Commands

## Start Development Server
```bash
npm run dev
```

## Run Research Queue (Inngest Dev Server)
```bash
npx inngest-cli@latest dev
```
This starts the Inngest dev server which processes background jobs for:
- Research document processing
- Sustainability analysis

## Run Supabase
```bash
supabase start
```

## Stop Supabase
```bash
supabase stop
```

## Supabase Status
```bash
supabase status
```
